# WhataHotel.com scrape — hotel dataset

Scraped from the live whatahotel.com city directories for 8 destinations
(Paris, Tokyo, Bali, Maldives, New York, London, Dubai, Maui).

## Files

| File | Contents |
| --- | --- |
| `whatahotel-hotels.json` | Full structured data — one object per hotel, with an `images` array |
| `whatahotel-hotels.csv` | Spreadsheet-ready — one row per hotel; `images` column is `\|`-separated |
| `whatahotel-hotel-images.csv` | One row per image (`city, hotelName, hotelID, imageUrl`) — best for bulk image work |

## Per-hotel fields

`city, name, hotelID, detailUrl, entryPrice, currency, priceRaw, thumbnail, imageCount, description, images[]`

- **entryPrice / currency** — the lowest "Rooms starting at" rate shown on the
  city directory (native currency: EUR, GBP, USD, JPY, IDR, AED).
- **detailUrl** — `https://whatahotel.com/booking/showRates.cfm?hotelID=<id>`
- **images** — gallery + featured image URLs at their original source resolution
  (brand-CDN images include `_original` / high-res variants).

## Coverage (370 hotels, 5,966 images)

| City | Hotels |
| --- | --- |
| Paris | 57 |
| Tokyo | 34 |
| Bali | 27 |
| Maldives | 24 |
| New York | 66 |
| London | 80 |
| Dubai | 70 |
| Maui | 12 |

- **245 of 370 hotels** have rich galleries (6+ images); 167 have 16+.
- **~93 hotels expose only one image.** This was verified (HTTP + headless
  Chromium render across all 8 cities): these hotels genuinely have a single
  featured image on whatahotel.com — it is **not** a JavaScript-hidden gallery.
  Brand-fed hotels (Four Seasons, Le Bristol, Ritz, etc.) carry full high-res
  galleries; many independents carry just one. So this dataset captures **100%
  of the images whatahotel.com publishes**.
- Star ratings, guest reviews, amenity lists, and per-room rate breakdowns are
  not published as extractable data on these booking pages (room rates come from
  a live availability call), so they are not included; `description` holds the
  page's meta description. Full galleries + amenities for the single-image
  hotels would need to come from each brand's own site or a hotel-content API
  (e.g. Amadeus), not whatahotel.com.

Generated by a Node crawler (city directories → detail pages), polite low
concurrency with retries.
